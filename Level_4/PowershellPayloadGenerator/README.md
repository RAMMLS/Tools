# PowerShell Payload Generator

## Описание
Проект демонстрирует принципы создания защищенных PowerShell скриптов для:
- Автоматизации задач безопасности
- Пентестинга в контролируемых средах
- Образовательных целей

## Требования
- .NET 6.0+
- Docker (опционально)
